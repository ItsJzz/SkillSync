## 🔑 Decision Tree

```
START
  │
  ├─ Is overall_score >= 50%?
  │   ├─ YES → "Ready to Advance" ✅
  │   │   └─ Continue with focused improvements
  │   └─ NO → "Keep Going" 📈
  │       └─ Show aggressive recommendations
  │
  ├─ For each TOPIC:
  │   ├─ Is topic_score >= 60%?
  │   │   ├─ YES → Add to "Strong Topics" ✅
  │   │   └─ NO → Add to "Weak Topics" ⚠️
  │   │       └─ Generate topic-specific learning path
  │   │
  │   └─ For weak topics:
  │       ├─ Find weakest area (quiz/sim/hands-on)
  │       ├─ Quiz weakest? → Recommend materials/reading
  │       ├─ Simulation weakest? → Recommend videos/playground
  │       └─ Hands-on weakest? → Recommend coding practice
  │
  └─ For each QUESTION TYPE:
      ├─ Is quiz_percentage < 60%?
      │   └─ YES → Add "Modular Learning" recommendation
      │
      ├─ Is simulation_percentage < 60%?
      │   └─ YES → Add "Visual Learning" recommendation
      │
      └─ Is hands_on_percentage < 60%?
          └─ YES → Add "Coding Practice" recommendation
```
